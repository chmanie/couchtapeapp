# couchtape

## Installation

1. Install MongoDB

On MAC-OSX use Homebrew:
```
$ brew install mongo
```

For Ubuntu/Debian hosts use:
```
$ apt-get install mongodb
```

2. Install the required Node_Modules with npm
```
$ npm install
```

## Starting the couchtape

```
$ node app
```

Open your browser with <code>http://localhost:3000</code>.
